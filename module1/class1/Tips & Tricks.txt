Visual Studio Code Editor Tips & Tricks
=======================================

1. Emmet for VS Code Editor
Type div>h1+p and ENTER key pressing will generate following code
    <div>
        <h1></h1>
        <p></p>
    </div>

2. Making code formatted
prettier makes non-formatted code prettier

Also, non-formatted code can be made formatted by command  
<div><h1>Heading</h1><p>Paragraph text</p></div>
    
Above non-formatted code will be formated like below if the code is selected and press > on command search and select "Format Selection"
    <div>
        <h1>Heading</h1>
        <p>Paragraph text</p>
    </div>
